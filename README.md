# Astra Keyboard

Astra Keyboard is our Android keyboard for fast Romanian typing and explicit,
server-backed voice dictation. Hold Enter to start recording, tap Enter to stop
and transcribe, or use Cancel to delete the local recording without upload.
Only the final transcription is inserted; the OpenAI credential remains on the
private server.

## Install and pair

1. Download the latest stable APK from Astra Settings.
2. Install and open Astra Keyboard.
3. Enter the one-time code generated by the Astra owner. The server address is
   already configured in the app; codes expire after ten minutes.
4. Grant microphone access, enable Astra Keyboard in Android input settings and
   select it.

Normal typing works without pairing. Voice dictation is authenticated by a
non-exportable Android Keystore key and is subject to per-device quotas.

## Scope

The product keeps the mature keyboard engine, dictionaries, layouts,
autocorrection, themes and password-field protections. Product-specific code is
kept around voice dictation, pairing, defaults and release identity. Build,
release and device procedures are in [RUNBOOK.md](RUNBOOK.md); architecture is
in [ARCHITECTURE.md](ARCHITECTURE.md).
